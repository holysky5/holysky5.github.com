Jquery 技巧 jquery

11 April 2015

自己整理的使用Jquery中常用到得代码段.常看看对自己的编程效率会有很高增长哦.

  • Jquery中对数组操作的支持

    $.merge(arr1,arr2,...); 
$.unique(arr1,arr2,...)
$.inArray(value,array,fromIndex);
$.map(array,callback);
$.grep(array,callback) ;
$.each()
$( 'div' ) .toArray()

  • 嵌套过滤

    el.filter(":not(:has(.selected))") //去掉所有不包含class为.selected的元素

  • is函数判断(非常有用) 检测当前元素是否符合某个selector

    el.is(":hidden")
el.is("[attr]")
el.is(".cls")

  • 检查selector 元素是否存在

     $(selector).length

  • 克隆

    $('#gbin1div').clone()

  • 判断文件是否加载完全

    $('#theGBin1Image').attr('src', 'image.jpg').load(function() {
    alert('This Image Has Been Loaded');
});

  • jquery 命名事件

    $('input').bind('blur.validation', function(e){
    // do what you want here
});

  • ajax同步 $.when( $.ajax('test1.html'),$.ajax("test2.html")) .done(fn).fail(fn) .then(successFunc, failureFunc ); //then可以成功函数与失败函数写在一起 .always(fn) //不管成功或者失败都执行

  • 全局ajax处理 $(document).ajaxXXX(); 对ajax进行拦截

  • jquery 绑定数据

    //The data method also accept namespaces
$('input').data('validation.isValid', true);

  • 强制打开新窗口

    $('a.popup').live('click', function(){
    newwindow=window.open($(this).attr('href'),'','height=200,width=150');
    if (window.focus) {newwindow.focus()}
    return false;
});

$('a.newTab').live('click', function(){
    newwindow=window.open($(this).href);
    $(this).target = "_blank";
    return false;
});

  • url处理

    //obj -> url
decodeURIComponent($.param(obj))
    form ->url
$('#form').serialize()
form ->array
$('form').serializeArray();

  • jquery 事件冒泡处理

    $("#test").click(function(e){
    return false; //阻止事件冒泡和默认行为并结束函数
});
$("#test").click(function(e){ 
    e.preventDefault(); //阻止浏览器默认行为
});
$("#test").click(function(e){ 
    e.stopPropagation();  //阻止事件冒泡
});
$("#test").click(function(e){ 
    e.stopImmediatePropagation(); //阻止剩余的事件处理函数执行并且防止事件冒泡到DOM树上。
});

  • jquery 复制对象

    // 浅层复制（只复制顶层的非 object 元素）
var new Object = jQuery.extend({}, oldObject);
// 深层复制（一层一层往下复制直到最底层）
var newObject = jQuery.extend(true, {}, oldObject);
